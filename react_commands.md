# React Commands – Complete Reference

---

## 1. Create React Project

### Using Create React App (CRA)

```bash
npx create-react-app my-app
cd my-app
npm start
```

Build:

```bash
npm run build
```

---

### Using Vite (Modern & Fast)

```bash
npm create vite@latest my-app
cd my-app
npm install
npm run dev
```

Build & preview:

```bash
npm run build
npm run preview
```

---

## 2. Run & Manage React App

```bash
npm start
npm run dev
npm run build
npm run preview
npm stop
```

---

## 3. Package Management (NPM)

Install package:

```bash
npm install package-name
```

Uninstall package:

```bash
npm uninstall package-name
```

Update package:

```bash
npm update package-name
```

Install globally:

```bash
npm install -g package-name
```

---

## 4. Common React Libraries

React Router:

```bash
npm install react-router-dom
```

Axios:

```bash
npm install axios
```

Redux:

```bash
npm install redux react-redux
```

Context API (built-in, no install needed)

---

## 5. ESLint & Formatting

Run lint:

```bash
npm run lint
```

Fix lint issues:

```bash
npm run lint -- --fix
```

---

## 6. Environment Variables

```bash
npm install dotenv
```

Run with env:

```bash
npm run dev
```

---

## 7. Clean Install (Fix Errors)

```bash
rm -rf node_modules package-lock.json
npm install
```

---

## 8. Testing (If Configured)

```bash
npm test
```

## 9. Git Commands (Common for React)
